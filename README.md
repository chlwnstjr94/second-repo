# Second Repo

## This is my second repo to practice how to start git with `$ git init`

## Commands

```shell
$ mkdir second-repo
$ cd second-repo
$ git init
// after creating remote repo on github,
$ git remote add origin {repo_url}

// After doing some work,

$ git add {fileName}
$ git commit

$ git branch -M main

$ git push -u origin main
```
